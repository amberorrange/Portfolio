Project

Backlog Tracker

Description: Backlog Tracker is a full-stack web app designed to help video game enthusiasts keep track of the games they want to play. The app allows users to search for games and add them to their backlog, which displays data such as the game’s title, description, genre, platform, etc. Games can be organized according to different filters and sorted according to user preference. The app provides a way to add reviews, where players submit the number or hours that they played a game. The app analyzes the data from reviews and uses it to display user statistics, which include the total number of hours played, the percentage they’ve played per genre or platform, along with other relevant information.

Tech stack used: PostgreSQL | SQLAlchemy | Python | Flask | Jinja | Javascript | React | jQuery | HTML | CSS | Bootstrap | Chart.js
